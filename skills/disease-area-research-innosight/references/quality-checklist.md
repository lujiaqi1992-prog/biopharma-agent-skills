## 数据完整性

- [ ] InnoSight 已先列出接口、再读取目标接口 schema，未猜测参数。
- [ ] 报告前已提供研究执行回执，列出实际 API/path、schema、查询口径、页数或 cursor 批次、结果量和缺口。
- [ ] 已覆盖流行病学、指南、获批药物、全球管线、临床试验、关键结果、中国注册、BD交易和近期新闻；未覆盖项已解释。
- [ ] 需要完整覆盖的数据已翻页至结束，并完成同义词/亚型补检。

- [ ] 中国本土企业布局已独立成节，不与跨国公司中国权益混淆。

## 临床与科学严谨性

- [ ] 每项关键临床数据包含人群、样本量、终点、数据截止日和来源。
- [ ] 疾病终点选择适配，不套用肿瘤指标。
- [ ] 单臂/早期试验没有被表述为确证性比较证据。
- [ ] 跨试验比较明确提示人群、设计和随访差异。
- [ ] 安全性同时呈现常见不良事件和关键严重/停药事件，不只写“耐受性良好”。

## 流行病学、市场与交易

- [ ] 流行病学数字注明年份、地域、病例定义和来源。
- [ ] 患病人数、确诊人数、可治疗人数和市场患者池未混用。
- [ ] 市场规模区分历史实际值、第三方预测和自主情景估算。
- [ ] 交易金额区分首付款、股权投资、里程碑、潜在总额和特许权使用费。
- [ ] 未披露信息明确写“未披露”，没有推测或重复加总。

## 引用与结论

- [ ] 关键数字、阶段、疗效、安全性、指南和交易均有就近引用。
- [ ] 优先引用一手来源；新闻稿或二手来源的关键结论已尽量交叉核验。
- [ ] 来源冲突已解释口径或采用理由。
- [ ] 事实、推断和战略判断清晰分开。
- [ ] 正文严格为五个一级部分，且技术展望可追溯到未满足需求和竞争证据。

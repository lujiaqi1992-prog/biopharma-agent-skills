# 交付前质量检查

## 范围与时效

- [ ] 已披露疾病、亚型、地域、创新药和中国本土企业口径。
- [ ] 已注明执行当天的数据截止日。
- [ ] 临床、审批与交易状态已用近期来源复核。
- [ ] BD 交易覆盖最近 24 个月，例外项已解释。

## 数据与连接器

- [ ] 已先发现可用工具与 schema，未猜测工具名、字段或参数。
- [ ] 若使用结构化连接器，已完整翻页并保存检索口径。
- [ ] 若连接器不可用，已使用公开一手来源降级，并披露覆盖限制。
- [ ] 管线已按别名、合作方和地区权益归一去重。
- [ ] 全球阶段、中国阶段、试验获批与实际启动已区分。

## 临床与科学

- [ ] 关键临床数据包含人群、样本量、终点、截止日和来源。
- [ ] 疗效指标适配疾病，未机械套用肿瘤指标。
- [ ] 单臂或早期试验未写成确证性比较证据。
- [ ] 跨试验比较已提示人群、设计和随访差异。
- [ ] 安全性包含关键严重事件与停药信息，不只写“耐受性良好”。

## 流行病学、市场与交易

- [ ] 流行病学注明年份、地域、病例定义和来源。
- [ ] 患病、确诊、治疗、可治疗人群和市场患者池未混用。
- [ ] 市场规模区分历史实际、第三方预测和自主情景。
- [ ] 交易金额区分首付款、股权、里程碑、潜在总额和 royalty。
- [ ] 未披露信息未被猜测或重复加总。

## 引用与结论

- [ ] 关键数字、阶段、疗效、安全性、指南和交易均有就近引用。
- [ ] 一手来源优先，新闻稿/二手来源已尽量交叉核验。
- [ ] 来源冲突已解释。
- [ ] 事实、推断、判断和未知清晰分开。
- [ ] 正文严格为五个一级部分。
- [ ] 建议对立项、BD 与 IP 监控有明确行动指向，且未冒充正式 FTO 或法律意见。
